## Productivity Dashboard

A modern **Productivity Dashboard Web App** built using **HTML, CSS, and JavaScript** to help users manage tasks, plan their day, and stay focused.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Features

* To-Do List

  * Add / Delete Tasks
  * Mark Important Tasks
  * Stored using LocalStorage

* Daily Planner

  * Time-based day scheduling
  * Persistent data storage

* Daily Goals Tracker

  * Add and complete daily goals
  * Clean card-based UI

* Pomodoro Timer

  * Start / Pause / Reset focus timer

* Weather Information

  * Displays live weather details

* Motivational Quotes

  * Dynamic quote display

* Theme Toggle (Dark / Light Mode)

  * User-preferred theme saved using LocalStorage

* Responsive Dashboard Layout

  * Horizontal scroll feature cards
  * Modern UI experience

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

##  Tech Stack

* HTML5
* CSS3
* JavaScript 
* LocalStorage API
* Weather API Integration
* Vercel (Deployment)
